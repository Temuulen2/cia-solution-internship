## Project Overview
This project contains the UI design for the CIA Internship LLC platform.  
The main flow starts from the **Project List page** and navigates into a **Project Detail page** when a project is selected.

1. User opens the Project List page  
2. User selects a project  
3. The Project Detail page appears
